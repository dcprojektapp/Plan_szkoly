---
description: Build Android APK for the Flutter application
---

To build the release APK file for Android:

1. Open a terminal in the project root.
2. Run the following command:
   ```bash
   flutter build apk --release
   ```
3. Once the build is complete, the APK file will be located at:
   `build/app/outputs/flutter-apk/app-release.apk`
